### HTML Tag

```html
<html>
    <title> HTML Web Page </title>
    
    <!-- Favicon Icon -->
     <link rel = "icon" type = "image/x-icon" href = "/images/favicon.ico">
</html>
```

1. Title

    Contains the title of the web page in the browser tab

1. Favicon

    - Small image displayed next to the page title in the browser tab
    - Save the favicon image in the images folder in the project root directory
    - A common name for favicon images is "favicon.ico"
    - `<link>` connects an external resource to the HTML document
    - `rel` means relationship and it tells the browser that the linked file is the icon for the webpage
    - `type` specifies the file type
    - `images/x-icon` means _.ico_ icon file
    - Other possible types of files are `images/png`, `images/jpeg` and others
    - `href` tells the browser where the favicon file is located. The path can be relative or absolute
